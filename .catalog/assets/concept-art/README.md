# Artes conceituais

Este diretório guarda mockups e artes de conceito para suporte de design e produto.

## Regra

- Conteúdo aqui não é arte oficial de runtime ou de bundle do aplicativo.
- Este material pode ser usado como base para criação futura de arte oficial.

## Convenção de nomes

- Use prefixo `concept-` nos arquivos (exemplo: `concept-icons-base-v1.png`).

## Não fazer

- Não referenciar arquivos desta pasta no `tauri.conf.json`.
- Não importar arquivos desta pasta diretamente para a UI de produção.
